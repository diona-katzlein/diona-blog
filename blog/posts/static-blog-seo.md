# SEO untuk Static Blog

Static blog tetap bisa SEO-friendly jika memiliki:

- Meta description
- Open Graph tag
- Sitemap XML
- RSS feed
- Struktur heading yang rapi

```html
<meta name="description" content="Deskripsi artikel kamu">
```
